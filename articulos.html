    Artículos sobre la Web

    Web 1.0

    La Web 1.0 es la versión estática, donde los usuarios solo podían leer contenido sin interacciones.

    En esta era, las páginas web eran principalmente informativas y no dinámicas.

    • Estático
    • Unidireccional
    • Sin interacción

    Web 2.0

    La Web 2.0 trajo consigo la interacción de los usuarios, las redes sociales y el contenido generado por los mismos.

    Se caracterizó por ser participativa y colaborativa.

    1. Interacción
    2. Contenido compartido
    3. Redes sociales

    Web 3.0

    La Web 3.0 es la era de la descentralización y la inteligencia artificial, con servicios personalizados.

    En esta fase, la web se hace más inteligente y centrada en el usuario.

    • Descentralización
    • Inteligencia artificial
    • Internet de las cosas
